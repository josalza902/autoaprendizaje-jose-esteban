La Metodología BEM

la Metodología BEM ayuda a organizar el codigo de css para que sea mas legible y reutilizable

bloque

El bloque es el componente más alto de la estructura. Representa una sección o componente independiente de la interfaz. El bloque es autónomo y no depende de otros elementos.

Características: Un bloque puede ser un componente de una interfaz, como un formulario, un botón, una barra de navegación, etc. Su nombre debe ser único y describe la función o el propósito del bloque.

elemento

El elemento es una parte del bloque que no tiene sentido por sí mismo y depende del bloque al que pertenece. Los elementos están ligados a un bloque y tienen una relación jerárquica con él.

Características: El nombre del elemento se forma a partir del nombre del bloque seguido de dos guiones (__). Un elemento no puede existir fuera del bloque que lo contiene.

Modifier (Modificador)

El modificador es un atributo adicional que cambia la apariencia, comportamiento o estado de un bloque o un elemento. Los modificadores permiten crear variaciones o diferentes estilos para un mismo componente.

Características: El modificador se utiliza para aplicar diferentes estados o estilos a un bloque o un elemento. El nombre del modificador se separa del nombre del bloque o elemento con un guion (--). El modificador puede expresar diferentes estados, tamaños, colores, etc.
