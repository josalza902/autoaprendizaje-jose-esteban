Combinadores en CSS

Los combinadores permiten seleccionar elementos en función de su relación con otros elementos.

  • Combinador de descendientes (espacio): selecciona elementos dentro de otro elemento.
  • Combinador de hijo (>): selecciona elementos hijos directos.
  • Combinador de adyacencia (+): selecciona el primer elemento que sigue inmediatamente a otro.
  • Combinador de hermanos generales (~): selecciona todos los elementos hermanos siguientes.

Combinadores Lógicos

Permiten combinar selectores para aplicar estilos de manera más flexible.

  • , (coma): Aplica un mismo estilo a múltiples selectores.
  • :is(): Selecciona múltiples elementos con menos código.
  • :where(): Similar a :is() pero sin afectar la especificidad.
  • :not(): Excluye elementos de un selector.

Pseudoclases

Permiten aplicar estilos a elementos en estados específicos.

  • :hover - Cuando el usuario pasa el cursor sobre el elemento.
  • :focus - Cuando el elemento recibe el foco.
  • :active - Cuando el elemento está activo.

Pseudoclases de Estructura

Se utilizan para seleccionar elementos en función de su posición dentro de un contenedor.

  • :first-child - Primer hijo de un elemento padre.
  • :last-child - Último hijo de un elemento padre.
  • :nth-child(n) - Selecciona elementos en base a una fórmula.
  • :nth-of-type(n) - Similar a nth-child pero basado en el tipo de elemento.

Reglas en CSS

Una regla CSS está compuesta por un selector y un bloque de declaraciones.

        selector {
            propiedad: valor;
        }

Ejemplo:

        p {
            color: blue;
            font-size: 16px;
        }
